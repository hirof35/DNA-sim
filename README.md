# 🧬 DNA Movable Simulator (TypeScript)

HTML5 Canvas と TypeScript を使用した、動的な DNA 二重らせんシミュレーターです。

## 🚀 使い方

1. リポジトリをクローンまたはダウンロード
2. パッケージのインストール: `npm install`
3. 開発サーバーの起動: `npx vite`

## 🛠️ 技術スタック
- TypeScript
- HTML5 Canvas API
- Vite (ビルドツール)
